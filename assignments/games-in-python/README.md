# 📘 Assignment: Games in Python

## 🎯 Objective

Students will learn to build interactive games using Python. They will practice string manipulation, loops, conditionals, and user input handling by creating a classic word-guessing Hangman game.

## 📝 Tasks

### 🛠️ Build Hangman Game

#### Description
Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. This will help you master string manipulation, loops, conditionals, and random selection in Python.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (_ _ _ format)
- Track incorrect guesses remaining
- End when word is guessed or attempts exhausted
- Display win/lose messages
